# Univa 3.0 — Scalable Educational Platform

## Загальний опис

Univa 3.0 — це перехід платформи на рівень масштабованої освітньої екосистеми.
Версія 3.0 фокусується на сервісній архітектурі, інтеграціях та розширенні каналів взаємодії з користувачами.

Платформа більше не є лише студентським workspace — вона стає інфраструктурною системою, здатною масштабуватись для великих навчальних спільнот та інтегруватись із зовнішніми сервісами.

---

## Основна мета версії 3.0

* Забезпечити масштабованість системи
* Розділити відповідальність між сервісами
* Розширити канали доступу до платформи
* Підготувати інфраструктуру для зростання кількості користувачів

---

## Service Architecture

У 3.0 відбувається винесення окремих сервісів із Core-застосунку.

### Notification Service

Окремий сервіс для:

* Email-сповіщень
* In-app повідомлень
* Push-нотифікацій
* Логування доставок
* Retry-механізмів

Сервіс працює асинхронно та обробляє події від Core.

---

### File Storage Service

Окремий сервіс для:

* Зберігання файлів
* Метаданих
* Контролю доступу
* Signed URL
* Підготовки до CDN

Core зберігає лише `file_id`, що забезпечує єдину точку істини.

---

### Telegram Bot Service

У 3.0 вперше з’являється Telegram-бот як окремий сервіс.

Функціональність:

* Перегляд розкладу
* Нагадування про дедлайни
* Повідомлення про нові файли
* AI-відповіді через Telegram
* Швидкий доступ до основних функцій

Бот не містить бізнес-логіки — він працює як адаптер до Core.
