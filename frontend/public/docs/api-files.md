# Файли

Файловий модуль Univa призначений для навчальних матеріалів, конспектів, презентацій, PDF і командних артефактів. Він повинен залишатися простим для користувача і передбачуваним для коду.

## Основні можливості

- завантаження навчальних файлів
- попередній перегляд поширених форматів
- прив'язка файлів до предметів або робочих просторів
- швидкий доступ із дашборду та групових модулів

## Рекомендована організація

- тримай окремі папки по предметах
- використовуй зрозумілі назви файлів
- позначай фінальні версії явно
- уникай дублювання одного документа в кількох копіях

## Що важливо в реалізації

- не ламай URL або формат метаданих файлу без потреби
- якщо додається новий preview-flow, інтегруй його в наявні UI-компоненти
- перевіряй, що діалоги перегляду не втрачають доступ до вкладених ресурсів

## Поширені сценарії

- спільна папка для лабораторних
- архів матеріалів семестру
- база лекцій для AI-конспектів
- зберігання презентацій і звітів для захисту
