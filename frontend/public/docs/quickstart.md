# Швидкий старт

Univa об'єднує розклад, дедлайни, файли, спільну роботу та AI в одному студентському workspace. Цей гайд допоможе швидко зайти в продукт і зібрати базовий навчальний простір за кілька хвилин.

## Що підготувати

- email або Google-акаунт для входу
- базовий список предметів на поточний семестр
- розклад пар або календар навчальних подій
- кілька навчальних файлів для першого workspace

## Базовий сценарій запуску

1. Створи акаунт і заповни профіль.
2. Додай розклад або імпортуй його з календаря.
3. Створи перші дедлайни для предметів.
4. Завантаж файли по курсам у спільне сховище.
5. Увімкни AI-помічника для конспектів і пояснень.

## Що отримаєш після налаштування

- єдину точку входу для навчальних задач
- календар із дедлайнами та нагадуваннями
- структуру предметів і матеріалів
- спільний простір для групових проєктів
- AI-інструменти для швидкої роботи з контентом

## Куди рухатись далі

Після першого входу переходь до розділів `Встановлення`, `Перші кроки` та `Інтеграції`. Якщо потрібен технічний огляд API, відкрий розділи блоку `API`.
