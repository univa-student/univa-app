# Швидкий старт

Ласкаво просимо до **Univa** — єдиної платформи для студентів.

## Що таке Univa?

Univa об'єднує розклад, файли, чати та AI-помічника в одному місці. Почніть за кілька хвилин:

1. Створіть акаунт через email або Google
2. Оберіть університет зі списку
3. Імпортуйте або додайте розклад вручну
4. Запросіть одногрупників

## Перше підключення

```javascript
const univa = new UnivaSDK({
  apiKey: "your-api-key",
  university: "knu-shevchenko",
});

// Отримати розклад поточного тижня
const schedule = await univa.schedule.get({ week: "current" });
console.log(schedule.lessons);
```

## Базовий план

Після реєстрації ви отримаєте доступ до всіх базових функцій **безкоштовно**.
