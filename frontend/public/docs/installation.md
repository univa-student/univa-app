# Встановлення

Univa побудований як Laravel API + React SPA. Для локального запуску проєкту потрібно окремо підняти backend і frontend, зберігаючи існуючу модульну структуру.

## Вимоги до середовища

- PHP і Composer для API
- Node.js і npm для frontend
- база даних, сумісна з конфігурацією Laravel
- сучасний браузер для React SPA

## Структура проєкту

- `api/` містить Laravel backend
- `frontend/` містить React застосунок
- модулі backend розміщені в `Modules/*`
- модулі frontend розміщені в `src/modules/*`

## Базовий запуск

1. Встанови залежності в `api/` і `frontend/`.
2. Налаштуй `.env` для API.
3. Запусти міграції та seed-дані, якщо вони потрібні середовищу.
4. Підніми Laravel-сервер.
5. Запусти Vite dev server для SPA.

## Що перевірити після старту

- авторизація через cookie працює стабільно
- frontend бачить backend без CORS-помилок
- маршрути `/dashboard` і `/docs` віддаються коректно
- основні модулі завантажуються без падінь
