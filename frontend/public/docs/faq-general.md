# Загальне

Цей розділ збирає короткі відповіді на найчастіші питання про платформу Univa, її структуру та базові сценарії використання.

## Що таке Univa

Univa — це студентська платформа для розкладу, дедлайнів, файлів, групової співпраці та AI. Вона зменшує кількість розрізнених інструментів і дає один навчальний workspace.

## Для кого призначений продукт

- для окремих студентів
- для навчальних груп
- для командних академічних проєктів
- для викладацьких або кураторських сценаріїв супроводу

## Яка головна цінність

Найбільша користь з'являється тоді, коли календар, матеріали, задачі та контекст спілкування не рознесені по різних сервісах. Саме цю фрагментацію Univa і прибирає.

## З чого починати новому користувачу

Почни з розкладу, дедлайнів і базових файлів. Після цього підключи інтеграції та AI, коли вже є дані, з якими їм можна працювати.
