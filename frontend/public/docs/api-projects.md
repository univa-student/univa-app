# Проєкти

У контексті Univa проєкти — це навчальні або групові робочі простори, де поєднуються люди, файли, задачі та комунікація. Саме тут модульність платформи дає найбільшу цінність.

## Що входить у проєктний простір

- учасники й ролі
- спільні файли та матеріали
- список задач або дедлайнів
- контекст спілкування для команди

## Коли варто створювати окремий простір

- для курсової або дипломної роботи
- для командного предметного завдання
- для факультативного дослідницького проєкту
- для спільного сховища матеріалів групи

## Принципи структурування

- один простір на одну чітку навчальну мету
- зрозумілі назви папок і артефактів
- дедлайни мають бути прив'язані до контексту проєкту
- документи не дублюються між кількома місцями без причини

## API-очікування

Будь-які зміни у відповідях для проєктів мають зберігати передбачувану форму. Frontend-композиції для груп, спільноти та файлових робочих просторів покладаються на стабільні сутності й поля доступу.
